# 主要功能

- 🔥代码生成：在弹窗中输入需求来生成代码
![Title](./resources/logo.png?raw=true&sanitize=true "Title")
- 📝Chat：在弹窗中输入问题进行对话
- ⌨️快速插入：在对话框中生成的代码，可直接点击快速插入到代码框对应的光标处
- 🔑快捷键：```Ctrl+Alt+Y```呼出代码生成，```Ctrl+Alt+U```呼出对话
